# Currency Converter

A simple currency converter web application built using **HTML, CSS, and JavaScript**.  
It fetches live exchange rates from the [Frankfurter API](https://www.frankfurter.app/) and displays the converted amount along with country flags.

## Features

- Select **From** and **To** currencies from dropdowns  
- Enter the **amount** to convert  
- Shows the **converted value** instantly  
- Displays **country flags** for the selected currencies  
- Handles unsupported currencies gracefully  

## Supported Currencies

- EUR (Euro)
- USD (US Dollar)
- GBP (British Pound)
- TRY (Turkish Liras)
- JPY (Japanese Yen)
- AUD (Australian Dollar)
- CAD (Canadian Dollar)

> Note: Only currencies supported by the Frankfurter API are available.

## How to Use

1. Open `index.html` in your browser  
2. Select the currencies and enter the amount  
3. Click **Convert**  
4. See the converted amount and the flags update
